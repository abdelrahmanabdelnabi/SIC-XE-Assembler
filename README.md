# SIC-XE-Assembler
An Implementation of the SIC/XE assembler written in Java

### Supported Features
- Sic-XE code with literals

### How to use
- Clone this project
    ```
    git clone  
    ```
- Build jar file
    ````
    make jar
    ````
- Move the code file 'code.asm' and the jar to the same dir

- Run Assembler using
    ````
    java -cp SIC-XE-Assembler sicXE.Asm code.asm
    ````
